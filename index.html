DealPal

DealPal is a simple Chrome extension that helps you track prices on Amazon, Walmart, and eBay product pages.

Click the DealPal icon while viewing a product detail page to scan and save the current price. Tracked items are stored locally in your browser so you can check back later and see changes.

You can support DealPal in two ways:

  • Shop on Amazon using the affiliate link inside the extension. Purchases help support development at no extra cost to you.
  • Donate directly via the Support button if you’d like to help keep the project going.
